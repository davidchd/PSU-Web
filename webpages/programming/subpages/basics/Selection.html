Selection 条件语句

A selection structures, also known as conditional structures, is used to make decisions based on the given condition. Programs have to make choices in order to have multiple route. For example, I want to use the program to calculate the difference between two numbers. It seems pretty simple, I just need to have the program use one number minus another. However, how am I going to know which number is greater and which is smaller? I will get a negative result if the program use the smaller number minus the greater number, which is obviously wrong. Therefore, we need to use selection statements to make choices.
条件语句用于让程序对给定的条件作出判断，计算机程序需要条件语句才能具备多种处理可能。举个例子，我希望用程序计算两个数之间的差。表面上看，这是个很简单的问题，只需要用一个数减去另一个数。但是如果在两个数未知的情况下，我怎么告诉程序第一个数比较大还是第二个数比较大呢？如果在程序中设定第一个数减第二个数，但给定的第二个数比第一个数大，那么就会得到一个负的结果。这显然是不正确的。所以，我们需要让程序具备判断的能力，可以根据不同的情况做出不同的决定。

In C++, there are several ways to let the program make a decision. Two most common ways are if statement and switch statement.
在C++中，有几种方式可以让程序根据条件做出选择，最常见的两种是 if 语句和 switch 语句。

If Structure if语句

The format of an if structure is given below.
if语句的格式如下。 


if (condition_1)
{
    // what to do if condition_1 is true
}
[ else if (condition_2)
{
    // what to do if condition_2 is true
}
else if (condition_3)
{
    // what to do if condition_3 is true
}
else if (condition_n)
{
    // what to do if condition_n is true
}
else
{
    // what to do if all the condition above is false
} ]

As you can see, an if statement may have minimum of 1 but up to unlimited number of conditions. I know it might be hard to understand. Here is a flow chart of the logic of if statement. Hope this can help you.
如你所见，if 语句可以有至少一个至多无限个的判断条件(conditions)。我知道这可能很难理解，希望下面的流程图可以帮到你们。

Here is an example of using if statement to convert a percentage score to letter grade.
下面是一个用 if 条件语句把百分比分数转换成字母成绩的例子。 


// parameter score stores the numerical score out of one hundred
char grade(int score)
{
    char result;
    if(score >= 90)
    {
        result = 'A';
        cout << "Well done!";
    }
    else if(score >= 80)
    {
        result = 'B';
    }
    else if(score >= 70)
    {
        result = 'C';
    }
    else if(score >= 60)
    {
        result = 'D';
    }
    else
    {
        result = 'F';
    }
    return result;
}

The last thing I want to make a note here is that the brace {...} in an if statement are optional if there is only one statement inside the brace. Thus, the above block of code can be simplified to below. They will work exactly the same.
最后，我想简单的提及一下大括号的省略。如果 if 语句中的某一个条件(condition)只对应了一条需要被执行的语句，那么这条语句前后的大括号就可以被省略。因此，上方的代码可以被简化如下，他们在运行效果上没有任何区别。 


// parameter score stores the numerical score out of one hundred
char grade(int score)
{
    char result;
    if(score >= 90)
    {
        result = 'A';
        cout << "Well done!";
    }
    else if(score >= 80)
        result = 'B';
    else if(score >= 70)
        result = 'C';
    else if(score >= 60)
        result = 'D';
    else
        result = 'F';
    return result;
}

Switch Structure switch语句

The format of an switch statement is given below.
switch语句的格式如下。 


switch(expression)
{
    case constant_1 :
       statements_1
       [break;]
    [ case constant_2 :
       statements_2
       break;
    case constant_n :
       statements_2
       break;
    default :
       statements_n ]
}

Similar to if structure, you can have as many cases as you want. However, one essential difference that you may need to keep in mind is switch will continue executing after it has finished the matched case until it reaches break;. Hope this flow chart can help you.
和 if 语句一样，你可以有无限多的cases。但是，这两种条件语句的一个核心的区别就是 switch 语句会在遇到符合的条件之后一直执行代码，直到遇到 break; 为止。而 if 语句只会运行第一个遇到的符合条件所对应的代码块。希望下面的流程图可以帮到你们。

Here is an example of using switch statement to output day of week from sequential number of week.
这是一个根据星期中第几天的数字来显示星期几的例子。 


// parameter month stores the sequential number of week
void displayWeek(int week)
{
    switch(week)
    {
        case 1:
            cout << "Sunday";
            break;
        case 2:
            cout << "Monday";
            break;
        case 3:
            cout << "Tuesday";
            break;
        case 4:
            cout << "Wednesday";
            break;
        case 5:
            cout << "Thursday";
            break;
        case 6:
            cout << "Friday";
            break;
        case 7:
            cout << "Saturday";
            break;
        default:
            cout << "Input Error";
    }
}

Maybe you have already figured out, both if and switch statements have its own advantages. If statement is more suitable for a scope of value or other conditions, where switch is easier to handle single value. You should choose which to use wisely based on the actual situation.
也许你们已经发现了，if 和 switch 语句其实各有优势。if 语句更适合处理范围性的数据和其他判断条件，switch 语句在处理有限可能的值时更简洁。对于不同的情况应该灵活的选择合适的条件语句。
