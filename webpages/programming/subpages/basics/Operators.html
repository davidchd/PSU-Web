Operators 运算符

In C++, there are several different kinds of operators.
C++中的运算符一般被分为几个类型。

  1. Arithmetic Operators 算数运算符
  2. Relational Operators 关系运算符
  3. Logical Operators 逻辑运算符
  4. Assignment Operators 赋值运算符
  5. Bitwise Operators 位运算符
  6. Other Operators 其他运算符

In this tutorial, I will only talk about the first 4 kinds of operators. They are the most commonly used operators and will be enough for programming beginners. Any operators will have an output and one or more input. Take X = Y + Z; for example; X stores the output of the addition operator, and Y and Z are the inputs of the addition operator. One thing I want to make note is that I used the variable name a for almost all of the following examples; however, in different examples, the variable type of a will vary.
在本教程中，我只会讨论前四种操作符。他们(大概)都是编程中最常用到的，对于编程初学者来说这些已经完全足够了。任何运算符都会有一个输出和一个或多个的输入。比如X = Y + Z;，其中X存储了相加运算符的输出，Y和Z是相加运算符的输入。另外，在下面的那些例子中，我都使用了相同的变量名(大部分是a)。但其实在不同的例子中，变量类型是会相应变化的，使用同样的变量名只是为了偷懒【小声。

Arithmetic Operators 算术运算符

Arithmetic operators can be used on any numerical based values. Types of variable that use arithmetic operators the most are probably integer and (double) floating-point.
算术运算符可以被用于任何数字值。使用算术运算符最多的变量类型大概是整型和(双)浮点型。

Operator 运算符 Name 名称 Description 描述 Example 例子
+ Addition Add two number together a = 4 + 8; a will be 12
- Subtraction Subtract the second number from the first number a = 12 - 4; a will be 8
* Multiplication Multiple the first number by the second number a = 3 * 4; a will be 12
/ Division Divide the first number by the second number a = 12 / 3; a will be 4
% Modulo Return the remainder of the first number being divided by the second a = 12 % 5; a will be 2
++ Increase Increase the number by one a = 10; a ++; a will be 11
-- Decrease Decrease the number by one a = 10; a --; a will be 12

Relational Operators 关系运算符

The outcome of relational operators are always boolean values, where the input can be very likely any given data type (as long as both sides of relational operators have the same type). Relational operators are mostly used in conditional (aka. selection) statements or repetitions. In examples below, I used a variable to store the result of relational operators. This is just for you to clearly understand what will be the output, but is not how programmers usually do.
关系运算符计算出的结果都是布尔类型。关系运算符在条件语句(selection statement)和循环(repetition)中最为常见。在下面例子中的表达式里，我用变量存储了关系运算的结果。这样做只是希望你们能更好地理解关系运算的结果，但并不是程序员通常的做法。(说实话，起码就我自己而言，很少用到布尔类型变量。)

Operator 运算符 Name 名称 Description 描述 Example 例子
== Equal to Return true if two values are the same a = 3 == 5; a will be false
!= Not equal Return true if two values are not the same a = 3 != 5; a will be true
< Less than Return true if the left side is less than the right side a = 3 < 5; a will be true
> Greater than Return true if the left side is greater than the right side a = 3 > 3; a will be false
<= Less or equal to Return true if the left side is less than or equal to the right side a = 3 <= 5; a will be true
>= Greater or equal to Return true if the left side is greater than or equal to the right side a = 3 >= 3; a will be true

Logical Operators 逻辑运算符

Both the input and the output of logical operators should be boolean type. Likewise relational operators, logical operators are most commonly found in selection statement or repetitions.
逻辑运算符的输入和输出都应该是布尔类型。和关系运算符相似，大部分逻辑运算符都出现在条件语句或循环中。

Operator 运算符 Name 名称 Description 描述 Example 例子
&& And Return true if both value are true, otherwise return false a = true && false;
a will be false (or 0)
|| Or Return false if both value are false, otherwise return true a = true || false;
a will be true (or 1)
! Not Return the opposite of the boolean value a = !true; a will be false

Assignment Operators 赋值运算符

We have talked a little about the most basic assignment operator in last section. However, there are many compound assignment operators as well. All compound assignment operators are optional; in other words, you can always use basic assignment operator and other operators to do the same thing as any compound assignment operators do. Further, you may have noticed, the last row of the table below is vacant. This means there are some other assignment operators which I did not show here. They are really useless (from my own point of view) and not friendly for beginners.
在上一节中，我们浅显的讨论了赋值符。但除了这个最基础的赋值符之外，C++中还有很多复合的赋值符。所有的复合赋值符都是可选的，换句话说，所有复合赋值符的功能都可以用基础赋值符和别的运算符结合来完成。除此之外，你可能已经注意到下面表格的最后一行是缺省的。这意味着有一些其他的复合赋值符不在列表中。我并没有把一些在我看来尤其没用的和对入门程序员很不友好的赋值符列出来。

Operator 运算符 Name 名称 Description 描述 Example 例子
= Assignment Assign the right side to the left side a = 7 + 8; a will be 15
+= Addition and assignment Add the right side to the left side a = 8; a += 5; a will be 13
(Equivalent to a = a + 5;)
-= Subtraction and assignment Subtract the left side by the right side a = 13; a -= 5; a will be 8
(Equivalent to a = a - 5;)
*= Multiplication and assignment Multiple the left side by the right side a = 3; a *= 5; a will be 15
(Equivalent to a = a * 5;)
/= Division and assignment Divide the left side by the right side a = 15; a /= 5; a will be 3
(Equivalent to a = a / 5;)
%= Modulo and assignment Assign the remainder of the left side being divided by the right side to the left side a = 12; a %= 5; a will be 2
(Equivalent to a = a % 5;)
