Operators List 运算符参照表

This is a table of most commonly used C++ operators. However, this is not a list of ALL operators in C++. In this table, operators are sorted based on the precedence. The smaller a precedence group number is, the higher precedence it has, and the earlier it will be executed. For more detail of each operator, please go to the Operator section in the left navigation bar.
