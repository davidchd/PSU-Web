Function 函数

A function in C++ is a group of statements that will be able to perform a specific task. As we discussed in the section of First Program, each C++ project must have at least one function, which is the main function. Programmers usually divide different tasks into different functions in order to make the program become easier to read and maintain. The format of creating a new function is given below. I will talk about how to call (run) a function in the next section.
C++中的函数是一组可以执行特定任务的代码。正如我们在 第一个程序 那一节讨论过的，任何一个C++的项目想要运行都至少要有一个函数(主函数)。程序员一般也会把每个程序划分成多个不同的任务，然后让每个函数执行不同的任务。这样做有利于增加程序的可读性并降低以后的维护成本。(其实我这样做的最主要原因是未来有可能需要用到类似的功能，可以直接使用以前写过的函数。)下面是创建一个函数的格式。我会在下一节中讨论如何调用(运行)一个函数。 


return_type function_name(parameter_list)
{
    function_body
}

In the declaration above, line 2 (the combination of return type, function name, and parameter list) is called function heading. Anything goes into the brace is called function body.
在上面的函数声明中，第二行(返回类型 return_type，函数名 function_name，和参数列表 parameter_list，这三者的组合)叫做函数头，剩下大括号中的内容叫做函数体。

Function Name 函数名

The restriction of a function name is mostly the same as variable name's. I will, anyway, list the enforced restrictions here. Of course, besides of these enforced requirements, good styles of naming a function or a variable is always encouraged.
函数名的命名规则和命名变量的相同。尽管如此，我还是列出了必须遵守的命名规则。除了这几条必须遵守的原则，好的命名习惯当然会让你的程序更上一步。

  1. A function name can only contain letters, and/or underlines (_), and/or numbers.
    函数名只能包括字母，数字，下划线(_)中的一种或多种。
  2. A function name cannot start with a number.
    函数名的第一个字符不能是数字。
  3. A function name cannot be the same as any C++ reserved word.
    函数名不能和任何C++关键词相同。

Parameter List 参数列表

The format of listing parameters is given below.
函数中参数列表的格式如下。 


[data_type parameter_name] [, data_type parameter_name] ...

As we discussed in the Variables section, a bracket [...] usually means optional item. Therefore, as we can see from the format above, parameter list is optional to a function. In other words, a function can have no parameter at all. A very common way of using a function without any parameter is to use the function to print some specific content to the screen.
正像我们在 变量 一节中讨论过的，中括号一般表示可选的内容。所以上面的格式就很容易理解了：一个函数可以有最少零个，最多无限个参数。你们可能会思考，如果没有任何参数传递到函数，函数有什么用呢？其实有用，比如一种非常常见的无参数函数就是向屏幕显示特定内容。

Return Type 返回类型

The return type of a function, also as known as function type, is like the data type of the output of a function. Some common return types are: int, double, bool, void, etc. One valid function has to have a return type, or a return type of void means no return. Once the function is assigned any return type other than void, there must be a line of code in function body to return a value of matched return type. I will use some examples to explain in detail.
函数的返回类型(也叫函数类型)，类似于函数的输出数据类型，一些常见的返回类型有整型 (int)，双浮点型 (double)，布尔型 (bool)，无返回型 (void)，等。一个函数必须具有返回类型，或者标明返回类型为空 (void)。当函数被标注了任何除了 void 的返回类型时，函数体 (function body) 中必须有至少一行 return 命令来返回与返回类型相匹配的数据。我将使用一些例子来详细解释。

Example 1 例子 1

Here is an example of a function used to output some specific contents to the screen.
这是一个用于向屏幕显示一些特定信息的函数。 


void display()
{
    cout << "This is a function " << "which will display these content.";
}

The result of calling this function is:
This is a function which will display these content.
调用这个函数的结果如上方斜体字。

The return type of this function is void, which means nothing will be returned. Thus, there is no return statement inside the function body.
这个函数的返回值是 void，意味着这个函数不会返回任何数据。所以在函数体内并没有 return 指令。

Example 2 例子 2

Here is an example of a function used to calculate the average of two numbers.
这是一个用于计算两个数的平均数的函数。 


double average(double num1, double num2)
{
    double sum;
    sum = num1 + num2;
    return sum / 2;
}

In this example, both the return type and two parameters are the same data type, double. As you can see, we declared another variable inside this function to calculate the sum of two numbers. However, this variable sum will not be able to be accessed from anywhere outside this function. Technically, every time this function is called, a new variable named sum will be created; and every time the function is finished, such a variable will be deleted.
在上面的例子中，函数的返回类型和两个参数都是 double 类型。我们在函数体中创建了一个新的变量 sum 来存储两个参数的加和。但是，这个变量是无法被从函数外的任何位置访问到的。实际上，每次这个函数被调用的时候，它都会创建一个新变量 sum，然后每次函数结束运行时从内存中删除这个变量。

Example 3 例子 3

Here is an example of a function being used as a timer. It will display how much time left in the timer, where time is passed to the function by time parameter.
这是一个类似计时器的函数，函数会显示计时器还剩余的时间。剩余的时间是通过 time 参数传递给这个函数的。 


void timer(int time)
{
    cout << "There are still " << time << " minutes left. ";
}

If the value of parameter time is 65, the result of calling this function will display:
There are still 65 minutes left.
假设 time 参数的值是65，那么调用这个函数的结果如上方斜体字。
