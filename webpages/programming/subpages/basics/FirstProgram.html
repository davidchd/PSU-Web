First Program 第一个C++程序

Code 代码 


#include <iostream>
using namespace std;
// mian function
int main() {
    // output
    cout << "Hello World.";
    return 0;
}

In this program, there are mainly two parts of code forming a complete program.
这个程序主要由两个部分组成。

The first parts is the first two lines.
程序的前两行属于第一部分。

The first line #include <iostream> is an include statement. In C++, there are many useful functions that have already been created as head files. This line of code declares that this program may use some functions from the <iostream> head file. Function is a relatively new concept, but you will be able to find links of function and predefined functions in the left navigation bar. <iostream> is one commonly used file in the C++ standard library. Any program has cin or cout statement must include <iostream> head file. A list of C++ standard head files can be found from IBM Knowledge center at here.
程序的第一行是一个include指令。C++语言定义了很多实用的函数(function)，并把他们打包成不同的头文件(head file)。这一行代码标志着此程序可能会用到<iostream>头文件中的一些函数。函数是一个比较新的概念，但在左侧的导航栏里有两个关于它的链接应该可以帮到你。C++的标准库中，<iostream>是一个非常常用的头文件。任何使用 cin 和 cout 的程序都需要包含<iostream>头文件。上面的链接是一份C++标准库中头文件的列表。

The second line using namespace std; is a declaration of namespace. Namespace is a very abstract concept even in the programming field. I may (or may not) talk about this later.
程序的第二行是对命名空间的声明。即使在程序领域，命名空间也是一个很抽象的概念。我以后可能会(也可能不会)再讨论它。(多半不会！)

The second part (line 3 - 9) is the rest of code except for the first two lines.
除了前两行，剩下的代码都属于程序的第二部分(第3 - 9行)。

Line 3 and 5 are comments. I will talk about this in the very next section. You can find the link for comments in the left navigation bar right after the link of this section.
第三行和第五行是程序的注释。我将会在下一节中讨论注释。在左侧的导航栏中，这一节下面紧挨着的链接就是关于注释的。

Line 4 to line 8 is the main function for this C++ project. Every C++ project must has a main function, and it has to be named this way. Once you execute an C++ project, program will start running here. int indicates this function will return an integer, as line 7 shows return 0;. main () shows the name of this function is "main". This is a reserved name in C++, which means any function named "main" will be recognized as the start point of this C++ project. And therefore, you cannot have two function named as "main". Further, things goes into {...} are the code which will really be executed.
第4 - 8行是这个C++项目的主程序。每个C++项目必须有一个主程序，并且必须按照这样命名。只要执行一个C++项目，程序就会从这里开始运行。第四行的开始，int是对于函数返回值类型的声明。在这里int代表该程序将返回一个整数，正如在第7行的返回指令中你所看到的。(在左侧的导航栏Data Type 数据类型的链接中你可以找到更多细节。)int 之后的main代表了这个函数的名字。在C++中main是一个保留关键字，任何名为main的函数都会被识别为C++的主函数，并作为这个C++项目的起始点。所以，在一个C++项目中只能存在一个名为main的函数。另外，在第4行和第8行两个大括号中间的内容是真正会被执行的代码。

In this very first program we have written, there are really only one line of code that will do some work, which is line 6. cout is an output statement in C++. It is not the only output command, but is very likely the most commonly used one. This is a built-in function in <iostream> head file. "Hello world." is the content that will be outputted.
在我们的这个程序中，只有一行代码(行6)是真正有意义的。这是一行输出指令。cout 并不是C++中唯一的输出指令，但很可能是最常用的一个。它属于<iostream>头文件中的内置函数。后面的"Hello world."是这条指令真正会输出的内容。

In C++, almost every executable command has to end with a semicolon ;. However, things that can be followed with a brace {...} usually do not need to have a semicolon at the end (such as functions, selections, repetitions, and etc).
在C++中，几乎每一行可以被执行的指令都需要以分号(;)结尾。但是，后面可以连接大括号({...})的指令一般不需要加分号(比如函数声明，条件语句，循环语句，等)。
