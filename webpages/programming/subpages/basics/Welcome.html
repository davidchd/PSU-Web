Welcome 欢迎

This is a tutorial for C++ programming.
这是一个C++程序语言的教程

The purpose of this tutorial is to help new programmers to better understand C++ programming concepts as well as its programing logic.
本教程的目的是帮助编程初学者们更好的理解C++程序语言的概念和程序逻辑。

For whom is this tutorial designed? 谁适用于此教程？

This tutorial is built for anyone who's trying to learn C++, no matter if you are trying to learn on your own or taking a programming course. Nevertheless, this tutorial may offer some extra help to Chinese students at Penn State as it provides Chinese translation for non-native English speakers.
本教程适用于任何想学习C++程序语言的编程者，无论自学还是作为编程课程的辅助材料。尽管如此，本教程可能会对中国学生提供一些额外帮助(如中文翻译)。毕竟相对于任何一种第二语言，大部分人都很可能会更擅长ta的母语。

As a Chinese student, I know the difficulties of learning a new programming language in English. When I first learned programming in English, I also faced some difficulties even though I had been an experienced programmer already. Therefore, I would like to try to help those with similar difficulties as I did.
作为一个中国学生，我深刻理解用英语学习一种新的程序语言的困难所在。当我第一次用英文学习编程时，虽然我已经是一个有不少经验的程序员了，但我仍然遇到了不少困难。所以我希望能帮助到可能遇到和我类似困难的同学。

What will this tutorial talk about? 本教程将会讨论什么？

This tutorial will discuss both algorithms and the implementation of the algorithms of C++. Understanding algorithms is always a difficulty for programming beginners. The concept of programming algorithms and logic are brand new ideas for those who have never programmed. I will be using some flow charts and pseudo code to explain the programming logic and using some examples or blocks of C++ code to explain the implementation.
本教程将会讨论C++程序的算法和算法的实现。对编程的入门者来说，理解程序算法一直是一个难题。程序算法和逻辑的概念对没有过编程经验的人来说将会是一种全新的观念。在本教程中，我将会使用流程图和伪代码来解释程序算法，并使用C++代码块来解释程序算法的实现。

However, I will not provide implementations for all algorithms. I believe that it is essential for programmers to think on their own and really understand the logic of programming instead of just copying codes and pasting them together.
但是，我并不会提供所有程序算法的代码实现。我相信理解程序逻辑和思考的过程对程序员来讲是非常重要的，而不是单纯的复制粘贴代码。
(自己寻找答案的过程非常重要！就算只是百度/谷歌搜索答案，也会对程序学习有很大帮助。)
