Data Types 数据类型

In C++, you will need to declare different kinds of variables to store different kinds of data. A variable means some reserved space in the computer's random access memory (RAM). A table of common data types in C++ is given below:
在C++中，你需要声明不同类型的变量来存储不同类型的数据。一个变量对应着计算机内存中的一些存储空间。下面的表中提供了一些常见的C++数据类型。

Type 类型 C++ Identifier C++标识符 Size 大小 Range 范围
Integer int 4 byte -2147483648 ~ 2147483647
Floating Point float 4 byte ± 3.4 e ± 38
Double Floating Point double 8 byte ± 1.7 e ± 308
Character char 1 byte -128 ~ 127 (or 0 ~ 255)
Boolean bool 1 byte true or false (0 represent false while any value other than 0 represent true)

In this table above, the size of a data type means the size of reserved space in RAM of this type of data, and the range of a data type is calculated based on its size. In computers, 1 byte equals to 8 bits, where 1 bit is the smallest storage unit in a computer. A bit is a binary storage which can be only 0 or 1. Thus, the range of a data type is 2 ^ (8 * size).
在上面的表格中，一种数据的大小是指这种类型的数据在计算机内存中所占用的空间。数据类型的范围是根据其大小计算的。在计算机领域中，一个字节(one byte)等于八位(eight bits)。位(bit)是计算机中最小的存储单位，它是二进制的存储器，只能存储1或0。所以，一种数据的范围就是2的(8*大小)次方。

Take Integer for example. The size of Integer is 4 byte, which is 32 bits. So there are 2 ^ 32 = 4294967296 possible numbers can be stored in a Integer. Therefore, its minimum number is -(4294967296 / 2) = -2147483648, and its maximum number is 4294967296 / 2 - 1 = 2147483647 (because of zero).
以Integer举例，它的大小是4字节，也就是32位。所以这种类型的数据一共可以存储2 ^ 32 = 4294967296种不同可能性的数字。因此，它的最小值和最大值就可以被算出来了。

One thing I want to mention here is string.
我想在这里先讨论一下string这种数据类型。

Maybe some of you who had some programming experience have already noticed. The data type of string is not listed in the table. String, also as known as character string, is a chain of characters. This kind of variable is usually used to deal with non-numerical values such as name of a person.
可能你们中曾有过编程经验的人已经发现了，string这种变量类型并没有列在上面的表中。字符串，本质上就是一串字符(char)。它是一种处理非数字信息最常用的数据类型。

Unlike those of types which are listed in the table, string is not a data type that is written in the core language of C++. Instead, it is more likely a user-defined data type in the C++ standard library. (And yes, you can define your own data types in C++.) In fact, this can be applied to many other programming languages as well.
不像上面表中列出的那些变量类型，string并没有被写在C++的核心语言里。它是作为一个预定义的数据类型被写在C++标准库中的。(没错！在C++中你可以定义自己的数据类型。)事实上，这也适用于很多种其他的程序语言。
