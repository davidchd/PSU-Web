Programming Environment 编程环境

In order to compile a C++ program, you will need a C++ compiler. A compiler is a kind of special software that translates the programming language into machine code. Currently, one of the most popular compiler for C++ is be the GNU Compiler Collection (GCC). It is a free set of compilers issued by Free Software Foundation (see References and Acknowledgement). The latest version of GCC can be found on
你将需要一个C++编译器来编译你的C++程序。编译器是一种用来把程序语言转换成机器代码的特殊软件。现在，最受欢迎的C++编译器之一是GNC编译器套件(GCC)。这是一套由自由软件基金会发行的编译器套装。最新版本的GCC见参考资料与鸣谢。

Furthermore, some web based C++ compilers are available; however, I recommend you to install a C++ compiler as well as an IDE.
一些在线的C++编译器也是存在的，但我并不推荐……最好还是安装一个本地的编译器和IDE叭 (我马上会解释这是什么)。
