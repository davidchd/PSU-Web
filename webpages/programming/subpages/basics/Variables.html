Variables 变量

Variables can be seen as the most core part in programming. As we mentioned on the previous section, variables are basically reserved space in the computer RAM. In C++, each variable must be assigned a type when it was created, and different types of variables will be reserved for different size of space. The format of creating a new variable is given below.
变量在编程中是可以说是最核心的部分。正如我们在上一个部分中提到的，变量就是计算机内存中的预留空间。每个变量在被创建的时候都必须指定类型，变量的不同类型会决定其在计算机内存中保留空间的大小。C++中声明一个新变量的语句格式如下。 


type variable_name(s) [ = value];

In the declaration above, type is stand for the data type of a variable. Variable_name is the name of this variable. Each variable needs a name to be located and further used. It is also possible to declare multiple variables of the same type in one declaration. In programming, and in this tutorial as well, a bracket [...] usually means optional item. The bracket here [ = value] means that it is optional to give an initial value to the variable when created.
在上面的变量声明中，type代表着这个变量的类型。variable_name代表着这个变量的名称。每个变量都需要一个名称来被定位，并在之后可以读取或者修改其中的数据。一行的变量声明中也可以声明多个同类型的变量。在编程界，中括号 [...] 通常用来表示可选的内容。(换句话说，中括号中的内容是可有可无的。)在变量声明的格式中，中括号中的 = value代表编程者可以选择在创建变量时赋予其一个初始值，但这不是必须的。

Variable Name 变量名

Good programmers tend to give meaningful variable names. In C++, variable names are case-sensitive, and there are some enforced restrictions of giving a variable name.
一个好的程序员一般会使用有意义的变量名。在C++中，变量名是区分大小写的。以下是一些命名变量时必须遵守的规则。

  1. A variable name can only contain letters, and/or underlines (_), and/or numbers.
    变量名只能包括字母，数字，下划线(_)中的一种或多种。
  2. A variable name cannot start with a number.
    变量名的第一个字符不能是数字。
  3. A variable name cannot be the same as any C++ reserved word.
    变量名不能和任何C++关键词相同。

Assigning Value 赋值

C++, and most of programming languages, uses a specific operator to assign values to variables. You may have already noticed from the declaration above, = is being used. However, the name of this operator is different from what we have learned in the algebra. In mathematics, we call it "equal sign"; but in programming, it is called assigning operator. We will discuss more operators in the next section.
C++和大部分其他程序语言中都是用一种特殊的符号给变量写入数据。你可能已经从之前的变量声明中注意到了，我们使用了 = 。注意，这个符号的名字和我们在代数中学到的并不一样。在数学领域，我们叫它"等于号"，但在计算机领域，它一般被称为"赋值符"(我会在下一节中详细解释各种操作符)。向一个变量写入数据的过程一般被称为对该变量赋值。

When a program runs to an assigning statement, it will first calculate the right part of the assigning operator, and then assign this value into the variable on the left side of the assigning operator. One thing that you may need to keep in mind is the data types of right side and left side of the assigning operator must match. I will use some examples in the rest of this page to explain.
当程序运行到一行赋值语句时，它会首先计算赋值符右边的部分，得到结果后，将右边表达式的值赋予到左边名称所对应的变量里。需要注意的是左右的数据类型必须对应，这里的"对应"不一定代表完全一样，我会在本页剩下的部分中分别讨论不同类型变量的赋值。

Integer Variables 整型变量

Integer variable, as it is named, can store integer numbers. Assigning an floating point number to a integer variable will result in an error. Some uses of integer variables are given below.
整型变量用于存储整数。把一个浮点数(小数)赋值给整型变量会引发编译器错误。下面是一些整型变量声明的例子。 


int a;          // declare one int variable
int b, c;       // declare multiple int variables
int x = 5;      // declare one int variable and assign initial value
int y = 10, z;  // declare multiple int variables and assign initial value to one of them

Floating-point & Double Floating-point Variables 浮点型/双浮点型 变量

Floating-point and double floating-point variables can store floating-point numbers. Assigning a floating point number to a integer variable will result in an error. However, assigning a integer number to a floating-point variable will NOT result in an error; the number will be stored with a fraction part of zero. Moreover, differences between floating-point number and double floating-point number do exist, but it will become obvious only if the program uses a significant number of variables (maybe over thousand times). Thus, we will only use double floating-point variables in this tutorial. I will also recommend you to do so because it is really unnecessary to distinguish float and double until you get into some huge projects. Some uses of double variables are given below.
浮点型和双浮点型变量可以存储浮点数(小数)。把整数赋值给浮点型变量不会产生编译错误，程序会自动把小数部分保存为零。浮点型变量和双浮点型变量的确有一些区别，但对于小体量的编程完全不会造成影响。在本教程中，我会统一使用双浮点变量用于浮点数的存储。我也推荐你们这样做，因为只有当你涉及到非常大的项目开发时，它们的区别才会显现出来。下面是双浮点变量声明的几个例子。 


double a;           // declare one double variable
double b, c;        // declare multiple double variables
double x = 5;       // declare one double variable and assign initial value (5 will be converted to 5.0)
double y = 9.5, z;  // declare multiple double variables and assign initial value to one of them

Character Variables 字符型变量

Character variables are usually used to handle non-numerical data. In C++, character data are indicated with single quotation marks ('...'). One single character with two single quotation marks surrounding is a character data (such as 'C'). Anything stored as a character data is a character, and therefore, it cannot be used for any calculation. For instance, int n = 3 + 5; is a line of valid code and will has a result of 8. However, int m = 3 + '5'; is very different. For some reasons (hidden content below), it will be compiled, but will not give you the right result. Some examples of declaring character variables are given below.
字符型变量通常用于处理非数字性的数据。C++中使用单引号('...')来表示字符型数据，被两个单引号夹在中间的单一的字符就是字符型数据。任何被存储为字符型数据的信息都只是字符，不可以直接参加计算。比如 int n = 3 + 5; 就是一行有效的语句，变量n将被赋值为8。但是，int m = 3 + '5'; 得到的结果就会有很大区别。因为一些原因，它可以正常运行(详见下一段最后的隐藏内容)；但这一行表达式并不会给出你希望的结果。下面是一些声明字符型变量的例子。 


char a;           // declare one character variable
char b, c;        // declare multiple character variables
char x = '8';     // declare one character variable and assign initial value ('8' is not stored as a number)
char y = '$', z;  // declare multiple character variables and assign initial value to one of them

This type of variable might be the one that will confuse you the most. Because this type of data is designed to store a single character, however, technically, such a data type actually falls into Integer category. I will make this part hidden by default. If you want to learn more detail you can click here to show or hide the rest content of this part.
这种变量可能是最难以理解的一类数据类型。这类变量是设计用于存储单一字符的，但是根本上讲，这类数据实际上属于整型(Integer)。我将会把接下面的一小部分内容设置为隐藏，如果你希望学习这部分更有难度的内容，你可以点击上方的"here"链接来显示或隐藏剩下的部分。
To begin with, any character, no matter it is a digit or letter or symbol, will have a matched numerical representation inside a computer system. Different software use different stands of encoding characters. The one that C++ usually use is called American Standard Code for Information Interchange (ASCII). You will be able to find a table for ASCII code and matched symbols on ascii.cl. In this table, the column of ASCII represent the number actually being stored in the computer system, where the column of Symbol represent the character that is displayed to the screen. C++ stores the ASCII code in a character data type and display it as a character. This makes it possible to assign either integer numbers or characters to character variables. Here is some examples. Hope this will not confuse you.
首先，任何字符在计算机中都有相对应的数字编码，无论是数字还是字母还是符号。不同的软件会使用不同的编码标准。C++中最常用的编码方式是美国信息交换标准代码(ASCII)。你将会在上方的链接中找到一个ASCII的对照表。在这个表中，ASCII一列代表了计算机中存储的实际编码数字，Symbol一列代表了屏幕上显示出的字符。C++的字符型数据实际存储着字符对应的ASCII编码，并将其显示为对应的字符。这就使得字符型变量可以被赋予整型数据和字符型数据。这里是一些例子，希望不会太难理解。 


char a = 'h';     // variable a now has a value of 'h'
char b = 104;     // variable b also has a value of 'h' because 104 matches character 'h' in ASCII
b = b + 1;        // variable b now has a value of 'i' because 105 matches character 'i' in ASCII
int num = 'h';    // just let you know, this is also a valid expression, and variable num will have a value of 104

Boolean Variables 布尔型变量

One boolean can only hold one of two possible values, it's either a true or a false. Both true and false are reserved words in C++. Sometimes true will be represented by 1 or any number other than 0, where false is represented by 0. Here are some examples of declaring boolean variables.
一个布尔类型变量只有两种可能的值：真(true)或者伪(false)，true和false都是C++中的关键词。下面是一些声明布尔类型变量的例子。 


bool a;             // declare one boolean variable
bool b, c;          // declare multiple boolean variables
bool x = true;      // declare one boolean variable and assign initial value
bool y = false, z;  // declare multiple boolean variables and assign initial value to one of them
bool i = 50;        // just let you know, this is a valid expression, and variable i will have a value of true (or 1)
